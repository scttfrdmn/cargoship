Get Started in Minutes

# Install CargoShip curl -L https://github.com/scttfrdmn/cargoship/releases/latest/download/cargoship-linux-amd64 -o cargoship chmod +x cargoship # Estimate costs before archiving ./cargoship estimate ./my-data --show-recommendations # Archive your data with optimal settings ./cargoship create suitcase ./my-data --destination s3://my-bucket/archives/
